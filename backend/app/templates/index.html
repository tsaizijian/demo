API 端點總覽

用戶認證

POST /auth/register - 用戶註冊
POST /auth/login - 用戶登入
POST /auth/logout - 用戶登出
GET /auth/profile - 獲取用戶資料

聊天室管理

POST /chatroom/create - 創建聊天室
GET /chatroom/list - 聊天室列表
POST /chatroom/join/<id> - 加入聊天室
POST /chatroom/leave/<id> - 退出聊天室

訊息系統

POST /message/send - 發送訊息
GET /message/room/<id> - 獲取聊天室訊息
DELETE /message/delete/<id> - 刪除訊息

私人訊息

POST /private/send - 發送私訊
GET /private/conversations - 對話列表
GET /private/messages/<user_id> - 私人對話

管理功能

POST /admin/room/<id>/kick/<user_id> - 踢出用戶
POST /admin/room/<id>/promote/<user_id> - 提升管理員
